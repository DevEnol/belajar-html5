Mengenal HTML

HTML merupakan bahasa markup standar untuk membuat halaman Web. Fokus dari HTML adalah untuk presentasi atau tampilan dari halaman web, sehingga tidak termasuk dalam kategori bahasa pemrograman. Karena itu HTML tidak memiliki fungsi seperti pada bahasa pemrograman (logika, repetisi, percabangan).

Membuat File HTML

Untuk membuat file HTML, gunakan applikasi text editor seperti: notepad atau notepad++ pada Windows, TextMate pada Mac, Sublime Text (www.sublimetext.com) untuk Windows, Mac dan Linux. Untuk melihat tampilan dari file HTML tersebut gunakan browser web seperti: Google Chrome, Mozila Firefox, dan Safari.

Daftar Contoh Tag HTML:

  • Tautan
  • Heading
  • Gambar
  • Daftar (List)
  • Tabel
  • Form Input
  • Karakter Entitas
